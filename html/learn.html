CSS Basics

This page contains visual examples for margin, padding, width, height, display, background, pseudo-classes and pseudo-elements. Each example is color-coded and commented to help you understand how CSS properties work.

1. Margin (space OUTSIDE an element)

Margin creates space outside the element's border. It pushes other elements away. Think of it as the element's personal space!

Margin on all sides (30px)
This box has equal 30px margin on all four sides.
Different margins per side
Top: 10px | Right: 40px | Bottom: 30px | Left: 40px
Margin Inline (50px)
This creates 50px margin on the LEFT and RIGHT sides.
Margin Block (40px)
This creates 40px margin on the TOP and BOTTOM sides.

2. Padding (space INSIDE an element)

Padding creates space inside the element, between the content and the border. It makes the element bigger and gives content breathing room.

Padding on all sides (30px)
Different padding per side
Top: 10px | Right: 40px | Bottom: 30px | Left: 40px
Padding Inline (60px)
This adds 60px of internal space on the LEFT and RIGHT.
Padding Block (50px)
This adds 50px of internal space on the TOP and BOTTOM.

3. Width & Height

Width and height control the size of an element. These properties work on block-level and inline-block elements.

4. Border

Borders create lines around elements. You can control their width, style, and color. Borders go between margin and padding.

Solid Border 2px thick, black color
Most commonly used border style
Dotted Border 3px thick, red color
Creates a series of dots around the element
Rounded Border 2px solid blue with 15px border-radius
Rounded corners make designs look modern and friendly

5. Display

The display property determines how an element behaves in the layout. It's one of the most important CSS properties!

Inline Display

6. Background Image

Background images can be added to any element. CSS gives you precise control over positioning, sizing, and repetition.

Note: Make sure your image path is correct relative to your CSS file!

7. Pseudo-elements

Pseudo-elements let you style specific parts of elements without adding extra HTML markup. They use double colons (::).

This paragraph demonstrates ::first-letter and ::first-line pseudo-elements. Notice how the first letter is larger and red, while the entire first line is bold and dark blue. Try selecting this text to see the ::selection pseudo-element in action with a yellow highlight!

8. Pseudo-classes

Pseudo-classes style elements based on their state or user interaction. They use a single colon (:).


Watch the button change colors as you hover and click. Orange -> Blue (hover) -> Red (click)

9. nth-child & Structural Pseudo-classes

Structural pseudo-classes target elements based on their position in the parent. Very useful for styling lists, tables, and menus!

  • First item : Green and bold (using :first-child)
  • Second item : Red and italic (using :nth-child(2))
  • Third item : Purple and underlined (using :last-child)

Each item is styled differently using structural pseudo-classes, no extra classes needed in the HTML!
